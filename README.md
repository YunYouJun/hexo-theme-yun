## Install

```sh
git clone -b master https://github.com/YunYouJun/hexo-theme-yun themes/yun
```

If you don't have jade & stylus renderer:

```sh
npm install hexo-render-pug hexo-renderer-stylus
```

### Plugins

```sh
npm install hexo-symbols-count-time
```

## Usage

```yml
# root/_config.yml
theme: yun
```
